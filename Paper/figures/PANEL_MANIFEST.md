# TrustCircuit redesigned-panel manifest

The current set is generated by `python -m scripts.paper_figures.generate_all` and
writes 35 independent vector PDFs to `Paper/figures/redesigned_panels_v2/`. The
manuscript, LaTeX, CSV files, logs, and legacy panel directories are not modified by
the generator. Every panel uses `matplotlib.rcdefaults()`, DejaVu Sans, a large
reference-matched canvas and typography, `text.usetex=False`, a tight bounding box,
one main plot area, and deterministic PDF metadata. No panel has an inset, panel
letter, raster image, or embedded chart note.

## Table-suited content

The following should remain tables rather than additional artwork: the full eight-cell
capability matrix behind F8a; exact ZK witness/key/proof/RAM/CPU values behind F2d;
exact DP rounding rows and composition values behind F3c/F3d; exact Native/VBS CPU and
RSS trial percentiles behind F4d; exact concurrency budget rows behind F5e; and exact
controlled configuration values/uncertainty behind F6d/F8b/F8d. The charts show trends,
composition, or comparisons; they are not a replacement for those exact tables.

## Cross-panel evidence notes

- CSV files are read-only figure inputs. Missing files, missing columns, or non-finite numeric values fail generation.
- Measured, locally measured, enclave-timer, and model-calibrated values retain their source meaning; model-calibrated circulation is not presented as a public-network measurement.
- PCHIP is used only as a monotone-shape-preserving visual guide through measured observations. Real measured markers remain visible in every smoothed panel.
- Output validation for this revision: exactly 35 one-page PDFs, no PNG output, no embedded raster images, and reproducible hashes after a second generation run.

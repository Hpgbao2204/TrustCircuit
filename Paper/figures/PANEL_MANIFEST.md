# TrustCircuit redesigned panel manifest

The redesigned set is generated with `python scripts/generate_redesigned_figures.py`.
It writes only the 28 independent vector PDFs in `Paper/figures/redesigned_panels/`.
The existing `Paper/figures/panels/` files and the manuscript are left untouched. All
panels use `matplotlib.rcdefaults()`, the default Matplotlib font, tight PDF bounding
boxes, and one primary plot area. No panel contains an inset, panel letter, or PNG.

## Cross-panel evidence notes

- `measured`/`locally_measured`/`measured_enclave_tsc` values are plotted as recorded; p50, p95, means, and bootstrap intervals are never regenerated from rounded paper values.
- `model_calibrated_from_measured_components` and model-calibrated circulation values remain visibly labeled and are not presented as direct measurements.
- The raw and processed CSVs are read-only inputs. The generator fails on missing files or required columns rather than silently fabricating a value.
- The output directory was checked after generation: exactly 28 one-page PDFs, no PNGs, and no embedded raster images.
